Discord Archiver
===========

Give me a config.json and create a directory:

```javascript
{
	"token"			: "--discordToken--",
	"directory"		: "archive/",
	"maxByte"		: 2097152,
	"channels" 		: ["general"],
	"formats" 		: ["jpg","png"],
	"blackList"		: ["testPerson"]
}
```

and I'll archive things for you.
